# BSU_7sem_Database
## Exchanges reference
1. Exchange Id (integer)
2. Exchange Name (text)
3. Formation Date (date)
4. Location (text)
5. Revenue (float)
## Companies reference:
1. Company Id (integer)
2. Company Name (text)
3. Formation Date (date)
4. Exchange Traded (reference to Exchanges)
5. Employees Number (integer)
